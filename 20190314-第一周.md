## A

Leetcode-7-Easy：

空间复杂度：O(1)
时间复杂度：O(log(x))

```java
class Solution {
    public int reverse(int x) {
        int result = 0;// 反转值
        while(x!=0){
            int left = x%10;
            if(x>0 && result > (Integer.MAX_VALUE-left)/10){
                // 如果是正数，则判断是否向上溢出
                return 0;
            }
            if(x<0 && result < (Integer.MIN_VALUE-left)/10){
                // 如果是负数，则判断是否向下溢出
                return 0;
            }
            result = result*10+left;
            x=x/10;
        }
        return result;
    }   
}
```
